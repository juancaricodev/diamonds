# Diamonds
Website Diamonds developed with Vue.js among other technologies.

## Stack
* Vue.js
* HTML5
* SCSS (SASS)
* JS

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
